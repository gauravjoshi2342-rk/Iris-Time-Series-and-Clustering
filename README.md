Iris Time-Series & Clustering Analysis 

Turning Unlabeled Data into Actionable Insights.
Project Overview :-
This end-to-end data science project is designed to investigate Unsupervised Learning and Temporal Pattern Recognition by analyzing the well-known Iris data-set with a non-conventional application (i.e. Clustering & Time-Series Analysis), demonstrating the value of extracting structure and trend-based knowledge from unstructured data. 

Technical Architecture
Data Processing: Implemented efficient data cleaning and numeric manipulation with pandas and numpy libraries. 

Unsupervised Learning:
K-Means Clustering to clusterize data points. Used Elbow method to analytically determine the ideal number of clusters. Time-Series Analysis: Applied time-series decomposition to isolate trend and seasonal effects from residual noise and provide deeper insights into the temporal dependencies of the dataset.
